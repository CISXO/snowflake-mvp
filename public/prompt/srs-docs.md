두 가지 프로젝트 방향 비교
A. 자영업 폐업 위험 예측

기존 소비 감소 추세 + 유동인구 감소 + 경쟁 업종 증가를 조합해 "위험 신호" 스코어링
시계열 데이터(2021~2025)가 있어서 코로나 전후 회복 패턴도 반영 가능
단점: 실제 폐업 여부 레이블 데이터가 없으면 지도학습이 어려움 (비지도 or 스코어링 방식으로 우회 필요)

B. 창업자를 위한 상권 분석

업종별 최적 입지 추천 (유동인구 × 소득 × 소비 패턴 × 경쟁 강도)
데이터만으로 완결성 있는 결과 도출 가능 → 해커톤에 더 적합
리츠 데이터를 "상권 신뢰도 지표"로 연결하면 차별화 포인트

분석 레이어 설계
Layer 1 - 상권 매력도 스코어링 (동 단위)

Layer 2 - 상권 리스크 경고 (폐업 위험 신호)

유동인구 YoY 감소 + 소비 감소 동시 발생 → "쇠퇴 상권" 경고
아파트 시세 급등 → "임대료 상승 위험" 경고
특정 업종 과밀 → "경쟁 포화" 경고

Layer 3 - 리츠 타이밍 인디케이터 (넥스트레이드)

해당 3개 구(서초/영등포/중구) 상업용 부동산 보유 리츠 종목 필터링
리츠 주가 추세 + 거래량 → "상업용 부동산 시장이 오르는 중 vs 하락 중"
결론: "지금 창업하면 향후 임대료 상승 압박 가능성 있음 / 낮음" 으로 표현
